## Tagesrhythmus (90-Tage-Plan)

| Phase | Pensum |
|---|---|
| Woche 1 (Setup) | 50 Agenturen recherchieren, 5 Test-Mails |
| Woche 2-4 | 20 Mails/Woche = 4/Tag |
| Woche 5-8 | 20 Mails/Woche + Follow-ups, erste Calls |
| Woche 9-12 | 30 Mails/Woche, Skalierung |

## Reflexions-Frage am Ende jedes Tags

> "Habe ich heute mit einem potenziellen Kunden geredet?"

Wenn nein an 3 Tagen in Folge → System ist gescheitert, Strategie reviewen.
